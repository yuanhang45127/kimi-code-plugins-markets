# Hello

When this skill is invoked, greet the user briefly and confirm the plugin is working.

Replace this file with your own skill. A skill is a Markdown file with YAML frontmatter:

- `name`: the skill name used to invoke it
- `description`: when the assistant should load this skill

The body contains the instructions the assistant follows once the skill is loaded.
